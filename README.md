 React Tours App

A simple and interactive Tours Listing application built using React and Tailwind CSS.  
Users can:

✔ View tour cards  
✔ Expand and collapse descriptions using **Read More / Show Less**  
✔ Remove tours using **Not Interested** button  
✔ Reload all tours using a **Refresh button**  
✔ See an empty state when all tours are removed  

This project is great for practicing **React hooks, props, conditional rendering, list rendering, and state updates**.



 Features

-  Display list of tours dynamically
-  Remove a tour from the list
-  Refresh tours when list becomes empty
-  Toggle description using Read More / Show Less
-  Clean and modern Tailwind UI
-  Built using functional components & hooks


 Tech Stack

- React (Vite or CRA)
- JavaScript (ES6+)
- Tailwind CSS
- useState Hook
- Props & Component Reusability
